# Sprint 1

El editor actualmente solo permite:

- Espacio de edición landscape.
- Tamaño fijo de espacio de edición.

Quiero que modifiquemos este proyecto para poder elegir el tamaño del espacio de edición, con el fin de que también se pueda diseñar para portrait usando un tamaño estándar de la industria. 

El proyecto debe poder arrojar un JSON con estas compatibilidades.

## Necesidades

Rediseña el espacio de trabajo para que sea compatible con un canvas vertical.
